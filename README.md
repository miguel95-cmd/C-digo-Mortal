# Código Mortal
Trabajo Colaborativo Contextualizado de Programación Orientada a Objetos

Una pequeña empresa dedicada a ofrecer servicios técnicos, como reparación de electrodomésticos, soporte IT y mantenimiento de equipos, enfrenta dificultades en la gestión de sus operaciones internas. Actualmente, la empresa no cuenta con un sistema adecuado para:

●	Gestionar información de clientes: No tienen un registro centralizado de datos de los clientes, lo que dificulta el seguimiento y la comunicación efectiva.

●	Organizar a los técnicos: Asignar técnicos especializados a tareas específicas es un proceso manual, propenso a errores y poco eficiente.

●	Controlar las órdenes de trabajo: No hay un método estructurado para registrar, monitorear y actualizar las órdenes de trabajo, lo que lleva a confusiones en los estados de los servicios (pendiente, en proceso, completado).

●	Establecer un sistema escalable: A medida que la empresa crece y diversifica sus servicios, su sistema actual no permite una fácil adaptación o integración de nuevas funcionalidades.

Descripció de la Solución Planteada:

La solución consiste en el diseño e implementación de un sistema de gestión integral, modular y escalable para la empresa, que automatice y optimice la organización de sus procesos internos. Este sistema estará basado en los principios de la Programación Orientada a Objetos (POO) y utilizará patrones de diseño para garantizar una estructura eficiente, flexible y adaptable.
